# Frankfurt CodeTalks

Website: https://fct.h-code.dev

A Programming study group for Koreans who living in Frankfurt.

If you have any suggestions for this study group, please feel free to leave your opinion on Issues.

## Development

1. Install packages

  ``` bash
  pnpm install
  ```

2. Start dev server

  ```bash
  pnpm dev
  ```

## Storybook

  ```bash
  pnpm storybook
  ```
